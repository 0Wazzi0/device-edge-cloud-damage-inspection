# 无人机车损勘察系统

这是一个部署在无人机端的智能车损勘察系统，能够在无网络环境下独立完成车损检测与勘察任务。

## 系统架构

本系统采用**无人机端独立决策架构**：

- **无人机端**：负责图像捕获、YOLO目标检测、位移判断和移动策略执行
- **完全离线运行**：所有决策均在无人机端本地完成，无需与基站通信

## 模块说明

### [yolo_service.py](file://d:\.code\Python3d11d8\project_whole_system\yolo_service.py)
- 负责车损检测的YOLO模型推理
- 提取车损中心点坐标
- 为决策提供数据支持

### [system_status.py](file://d:\.code\Python3d11d8\project_whole_system\system_status.py)
- 管理无人机的工作状态（搜索/勘察）
- 基于检测结果生成移动指令
- 控制无人机的勘察流程

### [main.py](file://d:\.code\Python3d11d8\project_whole_system\main.py)
- 系统主控制器
- 协调各模块工作
- 执行完整的搜索-勘察流程

## 功能特点

1. **完全自主运行**：无需外部通信，所有决策在无人机端完成
2. **实时检测**：利用YOLO模型快速检测车损
3. **智能定位**：自动调整无人机位置使车损处于画面中央
4. **状态管理**：清晰的状态转换逻辑确保勘察流程顺利执行

## 运行方式

```bash
python main.py
```

## 决策逻辑

无人机通过以下步骤完成勘察任务：

1. 进入搜索模式，扫描区域寻找车损
2. 检测到车损后切换到勘察模式
3. 基于车损位置计算移动指令，逐步调整位置
4. 当车损位于画面中央时完成勘察
5. 返回搜索模式或结束任务
